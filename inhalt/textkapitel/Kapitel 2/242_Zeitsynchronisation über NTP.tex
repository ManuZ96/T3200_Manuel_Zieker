% Zur Sicherstellung einer präzisen Zeitbasis wird die Systemzeit über das Network Time Protocol (NTP) synchronisiert. Dabei wird die aktuelle Uhrzeit von externen Zeitservern abgerufen und lokal im System gespeichert. Zusätzlich wird eine Zeitzonenkonfiguration verwendet, um die korrekte lokale Zeit inklusive Sommerzeitregelung darzustellen. Um langfristige Abweichungen zu vermeiden, erfolgt neben der initialen Synchronisation beim Start auch eine regelmäßige Nachsynchronisation.
